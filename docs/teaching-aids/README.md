# COLTIE GROW — Teaching Aids Index

This directory contains all teaching aids used across the 8-session program.

## Structure

```
teaching-aids/
├── slides/          # Slide decks (PPTX/PDF)
├── worksheets/      # In-class student worksheets
├── activities/      # Activity instructions & materials
├── handouts/        # Reference materials for students
├── rubrics/         # Evaluation rubrics
└── templates/       # Reusable templates
```

## Quick Reference

| Session | Key Teaching Aids |
|---------|------------------|
| S1: Welcome & AI Landscape | AI basics quiz, intro slide deck, AI diary template |
| S2: Prompt Engineering | COSTAR reference card, ethics checklist, prompt challenge cards |
| S3: Career Identity | RIASEC assessment worksheet, work values cards, AI career prompt guide |
| S4: University Research | 8-factor rubric, Reach-Match-Safety template, global comparison chart |
| S5: Personal Statement | Identity moments worksheet, essay before/after examples, sample essays |
| S6: AI Toolkit & Profile | No-code tools comparison, video profile planner, recording checklist |
| S7: Guest Speakers | Guest speaker brief, portfolio review rubric, presentation template |
| S8: Final Presentations | Certificate template, parent handout, alumni welcome pack |

## Slide Deck Naming Convention

`session-XX-topic.pptx` where XX is the session number.

## Worksheet Naming Convention

`topic-worksheet.pdf` — named by topic for cross-session reuse.
